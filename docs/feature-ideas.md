# Agent Panel 機能アイデア一覧

## ターミナル管理・レイアウト

1. **ペインのリサイズ** — ドラッグで各ペインのサイズを自由に変更（現状は均等サイズ）
2. **ペインの最大化/最小化** — ダブルクリックで1つのペインをフルサイズに展開
3. **ペインの分割方向指定** — 水平/垂直の分割方向をユーザーが選択可能に
4. **レイアウトプリセット** — 「2列」「3列」「メイン+サブ」などのプリセット
5. **ペインのタブ化** — 同じスロットに複数ターミナルをタブで重ねる
6. **ペインにカスタム名をつける** — ヘッダーをダブルクリックしてリネーム
7. **ペインのピン留め** — 重要なペインを誤って閉じないようにロック
8. **ペインのスワップ** — 2つのペインの位置を入れ替えるショートカット

## Claude CLI 連携

9. **会話履歴プレビュー** — トークン数・会話ターン数をヘッダーに表示
10. **全セッション一括停止/再開** — 全ターミナルに `/stop` を送る一括操作
11. **Claude モデル切り替え UI** — ペインごとに使用モデルを選択・表示
12. **プロンプトテンプレート** — よく使うプロンプトを保存してワンクリック送信
13. **マルチキャスト入力** — 同じプロンプトを複数ペインに同時送信
14. **セッションのエクスポート** — ターミナル出力をMarkdown/テキストに保存
15. **自動起動テンプレート** — `--model`, `--allowedTools` 等のオプションを事前設定

## 監視・ステータス

16. **ステータスインジケーター** — Claude が思考中/待機中/エラーかをアイコン表示
17. **完了通知** — Claude の応答完了時にデスクトップ通知
18. **コスト推定表示** — 各セッションのトークン使用量からコスト概算を表示
19. **レートリミットアラート** — 閾値到達時にワーニング通知（カスタマイズ可能）
20. **アクティビティログ** — 全ペインの操作履歴を時系列で確認

## UI/UX 改善

21. **ペインごとの色ラベル** — 背景色を変えて視覚的に区別
22. **フォントサイズ調整** — Cmd+/Cmd- でフォントサイズ変更
23. **ターミナル内検索** — 出力内を Cmd+F で検索
24. **最近のフォルダ一覧** — スプラッシュスクリーンに履歴表示
25. **プレビューサムネイル** — ペインが多い時にサムネイル一覧で俯瞰
26. **独立テーマ設定** — VS Code テーマと独立したテーマ選択
27. **ペインの独立ウィンドウ化** — ドラッグで外に出して別ウィンドウに

## ワークフロー自動化

28. **ワークスペース設定** — プロジェクトごとにターミナル構成を保存・復元
29. **スタートアップスクリプト** — 新規ターミナル作成時に自動実行コマンド
30. **タスクランナー連携** — VS Code Tasks と連携してビルド/テスト結果を表示
31. **Git ブランチ連動** — ブランチごとにセッションを自動グループ化
32. **スケジュール実行** — タイマーで指定時間後にプロンプト送信

## コラボレーション

33. **セッション共有** — セッションIDをチームメンバーと共有
34. **ペアプログラミングモード** — 複数ユーザーが同じペインを同時操作
35. **チーム用レートリミット表示** — チーム全体のAPI使用状況ダッシュボード

## 検索・ナビゲーション

36. **コマンドパレット** — Cmd+P でペイン移動・セッション検索
37. **ペイン番号ジャンプ** — Cmd+1〜9 で特定ペインにフォーカス
38. **出力のハイライト** — エラー・ファイルパス・URLを自動検出しクリッカブルに
39. **出力のdiff表示** — 2つのペインの出力を横並びで比較

## プラットフォーム対応

40. **Windows/Linux 完全対応** — ショートカット修正、キーチェーン代替実装
41. **SSH リモート対応** — VS Code Remote SSH 環境での動作サポート
42. **WSL 対応** — Windows Subsystem for Linux 環境での動作

## データ・分析

43. **セッション履歴ブラウザ** — 過去のセッション一覧を検索・復元
44. **使用統計ダッシュボード** — 日/週/月のClaude使用量をグラフ表示
45. **お気に入りセッション** — セッションをブックマークして素早くアクセス

## 高度な機能

46. **パイプライン/チェーン** — ペインAの出力をペインBの入力に自動転送
47. **条件付きアクション** — エラー検出時に別ペインで自動修正などのルール
48. **スニペット管理** — よく使うコード片やコマンドをサイドバーから挿入
49. **MCP サーバー管理UI** — MCP サーバー接続状態を表示・管理
50. **マルチモニター対応** — Agent Panel を別ウィンドウとして切り離し

## 優先度の高い候補

| 優先度 | # | 機能 | 理由 |
|--------|---|------|------|
| 高 | 6 | ペインのリネーム | 実装が軽くて UX 向上大 |
| 高 | 16 | ステータスインジケーター | Claude の状態が一目でわかる |
| 高 | 17 | 完了通知 | 複数タスク並列時に必須 |
| 高 | 2 | ペインの最大化 | 出力確認時に便利 |
| 高 | 37 | ペイン番号ジャンプ | キーボード操作が速くなる |
| 中 | 28 | ワークスペース設定 | 毎回同じ構成を作る手間を削減 |
| 中 | 12 | プロンプトテンプレート | 繰り返し作業の効率化 |
| 中 | 13 | マルチキャスト入力 | 複数プロジェクトに同じ指示 |
